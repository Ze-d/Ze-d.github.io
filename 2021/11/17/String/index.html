String

发表于2021-11-17|更新于2021-11-17
|阅读量:

操作方法

连接

concat(),使用concat()方法将新的字符串与旧字符串拼接后返回新的字符串

提取子字符串

slice(),substring()接受两个参数,字串的开始和结束位置

substr(),接受两个参数,字符的开始位置和提取的子串长度.

在处理负值的位置时:

  1. slice将所有负值转化为正值
  2. substr将第二个负值参数视为0
  3. substring将所有负值视为0

字符串位置方法

index()&lastIndex()

包含方法

startswith(),endswith(),includes()

trim()方法

trim(left/right)()用于清除字符串前后的空格之后返回一个副本

repeat()方法

接受一个整数参数为字符串复制的次数,返回拼接所有副本之后的结果

padStart()和padEnd()方法

接受两个参数,第一个参数是长度,第二个参数是可选的填充字符.

如果长度小于字符串的长度,则返回原始的字符串.

模式匹配方法

match()方法

此方法返回的数组和RegExp对象的exec()方法返回的数组是一样的.接受一个参数:正则表达式或者是regexp对象

search()方法

使用方法与match一样不过返回第一个匹配的位置索引,如果没有找到就返回-1.

replace()方法

接受两个参数,第一个参数是RegExp对象或者是一个字符串,第二个参数可以是一个字符串或者是一个函数.

如果第一个参数是字符串的话,要注意只会替换第一个匹配的字符子串,想要全部替换应该使用正则表达式.

第二个参数是字符串的情况下,可用特殊的字符序列,插入正则表达式操作的值.

split()方法

传入两个参数,第一个参数为分隔符,分隔符可以是字符串也可以是正则表达式,将正则表达式比配的子串作为分隔符,

如果匹配到开头或者结尾的话,会出现两个空元素.

第二个参数规定了返回的数组的长度.
