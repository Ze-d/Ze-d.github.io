Vue动画

发表于2021-12-17|更新于2022-03-22
|阅读量:

概述

Vue 在插入、更新或者移除 DOM 时，提供多种不同方式的应用过渡效果

过渡的类名

在进入/离开的过渡中，会有 6 个 class 切换

分别定义开始,生效,结束.

对于这些在过渡中切换的类名来说，如果你使用一个没有名字的 <transition>，则 v- 是这些类名的默认前缀。如果你使用了 <transition name="my-transition">，那么 v-enter 会替换为 my-transition-enter

使用方法

  1. 准备过渡的样式
  2. 使用transition标签包裹过渡的元素,配置name属性,如果有多个相同的动画,使用transition-group包裹并且给每一个元素指定key值.

使用第三方库

常见使用的库可以是Animate.css | A cross-browser library of CSS animations.
