##Emmet语法 ##

  1. 标签*数字，生成多个标签

  2. 标签a>标签b，生成包含的标签

  3. 标签a+标签b，生成兄弟的标签

  4. 直接使用标签./#，生成相对应的类和id的标签

  5. $用于生成自增数字序，例如.demo$5,则会依次生成标签

  6. 标签{默认文字}（*数字），生成默认文字

    快速格式化代码

  7. shift alt F格式化代码

  8. 设置保存时自动格式化代码

    复合选择器

    后代选择器

    元素1 元素2{样式说明}选择元素1的后代中的元素2
    可以嵌套选择后代；元素1和元素2可以是任意的选择器；元素2可以是元素1的任意后代

    子选择器

    元素1>元素2{样式说明}选择元素1的子代中的元素2
    元素2只可以是元素1的儿子

    并集选择器

    选择多组标签，同时定义相同的样式。

    元素1，元素2{样式说明}

注意最后一个选择器不要加逗号。

伪类选择器

元素：效果类别
  1. 链接伪类：a:link(为访问的)/visited/hover（鼠标经过的）/active（按下去时）注意：按照LVHA的顺序进行书写，链接一定要单独指定样式（因为链接有默认样式），实际开发中经常给a一个样式，再给a:hover一个样式。
  2. focus伪类：input：focus{}

    CSS的元素显示模式

    标签的显示方式；HTML分为块元素和行内元素；

    块元素

  3. 自己独占一行
  4. 可以修改长度，宽度，内外边距
  5. 宽度默认是父标签的宽度
  6. 是一个容器及盒子，里面可以放行内和块元素

注意：行内元素，文字标签内不可以放块元素。

行内元素

  1. 一行可以显示多个
  2. 高度和宽度无法直接设置
  3. 默认宽度就是内容宽度
  4. 行内元素只可以容纳其他行内元素和文本

注意：连接内不可再放置链接，a内可以防止块级元素，给a转化为块元素最安全

行内块元素

img input td同时具有两者的特点

  1. （行特点）一行可以放多个，但是会有间隙
  2. （行特点）默认宽度是内容的宽度
  3. 宽，高，内外边距都可以设置

元素模式显示的转换

一种模式需要另一个元素的特性，比如要怎加链接a的出发范围

  `a{display：block}/div{display: inline}/span{dispaly:inline-block}`

垂直居中技巧

使得行高=盒子高；line-height=height。

CSS的背景

  1. background-color：transparent|color；。默认为transparent
  2. background-image: none(默认)|url（）；。小的装饰性的图片或者是logo..便于控制位置
  3. background-repeat：repeat（默认）|no-repeat|repeat-x；
  4. back-ground-position：x y；x，y为坐标。参数为方位名词的时候，无所谓顺序；参数是精确单位：先x后y，单位为px，如果仅仅指定一个，该值一定是x坐标，另一个默认剧中；混合单位：先x后y，精确单位和方位名词混合使用。
  5. backgroun-attachment：scroll|fixed；
  6. 背景属性的复合写法：background：颜色 图片地址 背景平铺 滚动 位置。对顺序没有强制要求。
  7. 背景色半透明：background：rgba（0，0，0，0.3）a表示透明度
