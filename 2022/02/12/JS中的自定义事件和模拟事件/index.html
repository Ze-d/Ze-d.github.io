JS中的自定义事件和模拟事件

发表于2022-02-12|更新于2022-02-14
|阅读量:

自定义事件

自定义事件是创建一个之前没有的事件

Event()构造函数

let myEvent = new new Event(typeArg, eventInit);

typeArg是创建事件的名称;eventInit以字典的形式设置以下属性

  • “bubbles”，可选，Boolean类型，默认值为 false，表示该事件是否冒泡。
  • “cancelable”，可选，Boolean类型，默认值为 false， 表示该事件能否被取消。
  • “composed”，可选，Boolean类型，默认值为 false，指示事件是否会在影子DOM根节点之外触发侦听器。

CustomEvent()构造函数

let myEvent = new CustomEvent(typeArg, eventInit)

eventInit添加了新的属性detail,传入一个对象,提供事件关联的相关值

使用方法

对于原生的事件,通过原生的操作就可以直接触发,而对于自定义的事件,要通过代理到一个原生的事件上再dispatchEvent().

模拟事件

老版本方式document.createEvent()

模拟用户的原生操作.再document对向上使用createEvent()方法.

var event = document.createEvent(type);

type 是一个字符串，表示要创建的事件类型。事件类型可能包括"UIEvents", "MouseEvents", "MutationEvents", 或者 "HTMLEvents"。

注意这里的initevent:对于不同的事件类型对象,其初始化的方法名也不同: initMouseEvent(),initKeyEvent(),initCustomEvent()

新版本方法:创建事件类型对象

构造函数:mouseEvent();keyboardEvent();CustomEvent().

等于说将老版本的方法中:创建事件和事件初始化合并到了构造函数中
